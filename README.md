# 🚦 Road Accident Analysis Dashboard in Excel

## 📌 Project Overview

This project presents an interactive Road Accident Analysis Dashboard built using Microsoft Excel. The dashboard transforms raw accident data into meaningful insights by analyzing casualty trends, vehicle involvement, road conditions, and accident severity.

The project demonstrates practical skills in data cleaning, data analysis, dashboard design, KPI reporting, and data visualization.

---

## 🎯 Objectives

- Analyze accident and casualty trends.
- Compare current year and previous year performance.
- Identify high-risk vehicle categories and road types.
- Understand monthly accident patterns.
- Provide interactive reporting for data-driven decision-making.

---

- Microsoft Excel
- Pivot Tables

- Urban vs Rural Areas

- Weather Condition
- Accident Severity

## 📈 Key Insights

- Identified the vehicle categories contributing to the highest number of casualties.
- Analyzed monthly trends to identify seasonal patterns.
- Compared accident statistics across different road types.
- Examined casualty distribution under varying lighting conditions.
- Evaluated accident occurrences in urban and rural areas.
